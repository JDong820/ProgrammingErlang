Basic Concepts
==============

* erl has no annotations, i.e. -module, -export, etc.
* Emacs hotkeys in erl:
<pre>
^A
^D
^E
^F
^B
^P
^N
^T
</pre>
* _ 'anonymous variable'
* [car|cdr] lists
* $ = ord()
* \x{hex} for unicode
* `f()` 'forget' all variable bindings, similar to octave `clear` command.
* `help()`, not `help`.
